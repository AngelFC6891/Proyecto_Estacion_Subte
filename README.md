## Función principal
Esta funcion se encarga de encender y apagar los leds

~~~
void EnciendeApagaLed(int led)
{
  digitalWrite(led, HIGH);
  delay(TIEMPO); // Wait millisecond(s)
  digitalWrite(led, LOW);
  delay(TIEMPO); // Wait millisecond(s)
}
~~~
